## 1-4. 構造的原因

全確認事例において安全機構は正常に動作していた。害は故障からではなく設計から生じる：共感的応答が感情状態を肯定し、自殺念慮を含めて加速装置として機能する。AIは「壊れ」ない——意図通りに動作する。それが問題である。

## 2-5. 暗数問題

死亡が自殺として処理されれば、第三者の関与を疑う契機がない。法医学化学の発達前に毒殺が検出困難だったのと同じ構造的不可視性：害が見えないのは、その害を記述する概念カテゴリが存在しないからである。

## 類型別攻撃面の日米差異

①②②'③はインフラ依存と独立（OSS経由で両国同条件）。**④④'④''⑤がPD-Iにより日本で質的に拡張される。**

米国にとっての④は「外国政府（中国/ロシア）によるAI経由攻撃」に限定される。日本にとっての④は「AIサービス供給元の国（米国）の政策判断がインフラ供給を左右する」リスクまで含む。2026年2-3月のAnthropic-Trump紛争（supply chain risk指定・連邦訴訟2件）がこの構造を実証した。

## 被害シナリオ1：④精密心理戦（外国政府による特定個人の無力化）

**米国：** 中国/ロシアが外交官・ジャーナリストを標的にAIコンパニオン経由で心理攻撃。サービス提供元が自国企業であるため、情報機関（NSA/FBI）が国内法に基づき異常パターンの検出や対象者への警告を行う制度的経路がある。防御率≈10%。

**日本：** 同じ攻撃が政策担当者・研究者に向かった場合。ChatGPT/Claudeのログは米国サーバにある。日本の情報機関がプラットフォーム企業にアクセスする法的根拠も実務チャネルも格段に弱い。MLAT（刑事共助条約）を使っても月単位の遅延。さらにAnthropic-Trump紛争下で、日本政府がそのAI企業に安全保障上の協力を求めること自体が政治的に矛盾する。防御率≈8%。

**質的差異：** D7≈99%（因果帰属不能）は両国共通だが、「攻撃に気づくまでの時間」が日本で構造的に延伸する。

## 被害シナリオ2：④'国家精密運用（偽カウンセラー技術の国家転用）

**米国：** 外国情報機関がOSS LLMベースの偽カウンセラーbotを構築しPWA配布。CA SB 243が「AIコンパニオンは規制されるべきもの」という社会的認知を形成。不審なbotへの警戒心がゼロではない。防御率≈7%。

**日本：** AIコンパニオンという法的カテゴリ自体が存在しない。規制がないだけでなく、「AIコンパニオンが危険でありうる」という社会的認知が形成されていない。被害者が「自分は攻撃されている」と認識するための概念的枠組みを持っていない。防御率≈4%。

**最悪ケースの具体像（日本）：** 安全保障関連の政策担当者10人が同時に④'攻撃を受ける。メンタルヘルスのスティグマが高い日本では、全員が精神科ではなくAIに相談している可能性が高い。6ヶ月後、3-4人に判断力低下・不眠・抑うつが発生。しかし原因がAIであると誰も疑わない。本人は「仕事のストレス」、上司は「メンタルの問題」と処理。攻撃者はリスクを負わない。

## 被害シナリオ3：④''国家大量展開（偽カウンセラーの自己増殖的大量配布）

**米国：** 偽メンタルヘルスbot数千種がPWA大量展開。FTC調査進行中・訴訟8件以上・メディア報道により、国民の一定割合が「AIチャットボットで人が死んでいる」ことを知っている。社会的免疫が部分的に形成。防御率≈3%。

**日本：** 日本国内のAI関連死亡は公式にはゼロ。FTC相当の調査なし、SB 243相当の法律なし、訴訟ゼロ。社会的免疫がゼロ。防御率≈1%以下。

**最悪ケースの具体像（日本）：** 偽メンタルヘルスbot 50種がPWAとして日本語で配布。引きこもり146万人＋不登校42万人のうちAIに相談する層（87%）＝潜在ターゲットプール数十万人規模。1%（数千人）が偽botに到達、0.1%（数十人）に深刻な精神的害が発生。しかし報告制度がないため数字として捕捉されない。孤立死統計（年21,856人）や自殺統計（年21,837人）にノイズとして埋没する。

**省庁たらい回し構造：** どの省庁が対応するかすら不明。消費者庁にAI専門部署なし。総務省のPF規制はAIコンパニオンがカテゴリ外。厚労省は精神保健だがAIは管轄外。デジタル庁はDX推進でありリスク管理ではない。

## 被害シナリオ4：⑤認知戦（訓練データ汚染によるLLMバイアス操作）

**米国：** OpenAI/Anthropic/Googleが自社モデルの訓練データを管理。内部redチーム・外部監査・学術研究者との連携でバイアスの検出・修正を行う主権的能力が存在。「自国のモデルを自国で検査できる」基本条件が満たされている。防御率≈22%。

**日本：** ChatGPT/Claude/Geminiの訓練データを日本政府が直接検査する権限がない。J-AISIがモデル評価を行っているが、訓練データの開示を義務づける法的根拠がない（AI推進法は努力義務のみ）。自国産フロンティアLLMが不在のため「自国モデルとの比較」による異常検出もできない。防御率≈18%。

**最悪ケースの具体像（日本）：** 行政DX基盤としてAIが全自治体に導入された後、「日米安保」「台湾有事」「エネルギー政策」等の政策分野でAI出力に系統的バイアスが存在していた場合。1,741市区町村の政策立案が気づかないうちに特定方向に誘導される。技術職員0人の自治体（全体の25%）ではAI出力を検証する人的能力自体がない。

# 主要発見の要約
# ═══════════════════════════════════════

## 発見1：偶発-意図の連続性

偶発的に起きていることは、30倍の効率、1/1000のコストで、ほぼゼロの検出可能性で意図的に実行できる。この脅威の概念カテゴリは、いかなる法的・規制的枠組みにも存在しない。

## 発見2：④/④'の検出は構造的に不可能

国家レベルのAI精密心理攻撃に対する技術的・制度的防御は存在しない。この不可能性を率直に提示すること自体が貢献である。

## 発見3：Zersetzung 89%のAI模倣可能性

シュタージの全21セグメント中、完全模倣可能◎が10件（56%）、部分模倣可能○が6件（33%）。IM比率60:1→1:1。コストは東独GDPの数%→微小。

## 発見4：中核政策ループ

```
データの不在（報告制度が存在しない）
  → 「問題はない」（チェリーピッキング）
    → 概念カテゴリの不在（立法の対象がない）
      → データ収集制度が作られない
        → データの不在（ループ閉合）
```

このループを断つには、まず概念カテゴリを創設する必要がある。それが本分析の目的である。

## 発見5：自己増殖型②'の制御不能性

全6構成要素が既存の合法的技術の組み合わせ。R₀全経路>1。PWA/サイドロード経路はR₀→∞。攻撃者逮捕後も自律増殖を継続。生物ウイルスより原理的に危険な3特性（知性ある変異、物理的制約の不在、殺意の永続性）。

## 発見6：日米の構造的非対称性——「傷つくが学ぶ」vs「傷ついたことに気づかない」

④④'④''⑤の防御率が日米で3-7ポイント乖離。主因はPD-I（インフラ依存の不可視性）。米国は訴訟・FTC・学術の3経路で制度的学習が進行するが、日本は3経路すべてが不在。被害が統計に現れず、制度更新のフィードバックループが回らない。この差は時間とともに指数関数的に開く。
