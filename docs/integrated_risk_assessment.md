## 評価フレームワーク（固定ルール）

### 脅威度の計算式
```
脅威度 = 0.30×影響規模 + 0.30×不可逆性 + 0.25×致死性 + 0.15×近接性
```

### 緊急度の計算式
```
緊急度 = 脅威度 × (1 - 認知度/10) × (1 - 政策対応度/10)
```

### 信憑性に関する注記

| ❌ しないこと | ✅ すること |
|---------------|-------------|
| 起きる/起きないの予言 | 相対的危機度の比較 |
| 正確な発生確率の提示 | 優先順位の判断支援 |
| 科学的証明としての使用 | 思考整理・議論の出発点 |

## 発見事項

### パターン1：緊急度の層別分布

| 層 | 緊急度平均 | 解釈 |
|----|-----------|------|
| AGI準メタ（上流〜中流） | 67.8 | **最高**。開発競争が認知・対応を上回る |
| 合成生物学B | 68.3 | **最高**。技術的に最も切迫 |
| 神経支配B | 58.7 | 中。商用化前が対策窓口 |
| 認識論崩壊B | 47.3 | 低（親リスクの認知度が高いため） |
| 創発人材B | 55.3 | 中。「問題」と認識されにくい |
| ハブ衛星 | 46.5 | 低（脅威度は高いが認知あり） |

### パターン2：「脅威度は高いが緊急度は低い」リスク群

| リスク | 脅威度 | 緊急度 | 理由 |
|--------|--------|--------|------|
| A/P紛争 | 82.75 | 52 | 認知度2.5、一定の議論あり |
| ミーム兵器 | 79.25 | 46 | 認知度4.5、フェイクニュース議論あり |
| AI認識論崩壊（親） | 70.25 | 43 | 認知度6.0、広く認知されている |

→ 「認知されている」リスクは緊急度が下がる。本プロジェクトの焦点は「低認知×高脅威」のリスク。

### パターン3：バンドル横断的な侵食構造

```
AGI準メタ層の全リスク
    ↓ 加速・変質
基盤リスク層（意味喪失 → 意味特異点へ接近）
    ↓ 脆弱性生成
3バンドルすべて + ハブ衛星リスク
    ↓ 表面化
個別のカタストロフ（バイオテロ、ミーム兵器、BCI攻撃…）
```

上位層のリスクが下位層の対策能力を削ぐ構造。これが「派生リスクのほうが緊急」という逆転現象の原因。

### パターン4：対策の構造的差異

| 層 | 対策の性質 | 有効期間 |
|----|-----------|---------|
| AGI準メタ | ガバナンス・規制（上から設計） | AGI出現まで |
| 基盤リスク | 存在的基盤層（下から育てる） | ASI時代まで（それ以降は不明） |
| 合成生物学B | 経済的インセンティブ（攻める意味を消す） | 持続的 |
| 認識論崩壊B | 存在的基盤層0-6 | ANI/AGI時代は有効 |
| 神経支配B | 神経的権利（法的枠組み） | 商用化前が窓口 |
