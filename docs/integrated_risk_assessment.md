## 評価フレームワーク（固定ルール）

### 脅威度の計算式
```
脅威度 = 0.30×影響規模 + 0.30×不可逆性 + 0.25×致死性 + 0.15×近接性
```

### 緊急度の計算式
```
緊急度 = 脅威度 × (1 - 認知度/10) × (1 - 政策対応度/10)
```

### 認知度の加重平均
```
認知度 = 一般×0.5 + 学術×0.25 + 政策×0.25
```

## 信憑性に関する注記

> **本リスク表は、将来リスクの相対比較と議論整理を目的としたものであり、
> 発生確率や時期を確定するものではない。
> 確度Cリスクは推測的シナリオとして明示的に扱う。**

### 信頼できる部分
- ✅ 定性的構造分析 - リスク要素の因果関係
- ✅ リスク間の相対比較 - どれが最も危険か
- ✅ 現実事象の観察と記述 - 既に起きていることの記録

### 限界がある部分
- ⚠️ 新規リスクのスコア - 既存リスクより推定誤差が大きい
- ⚠️ 認知度・前兆度 - 新規リスクは観測事例が少なく、推定に依存
- ⚠️ 緊急度の絶対値 - 相対比較用の便宜的指数

### 緊急度と確度の関係について

本フレームワークでは、**確度が低いリスクでも緊急度が高くなる場合がある**。
これはバグではなく意図的設計である。

| 問い | 対応する指標 |
|------|--------------|
| 「起きるかどうか」 | 確度（A/B/C） |
| 「準備すべきかどうか」 | 緊急度 |

確度Cであっても「認知されていない × 政策対応なし」であれば、**準備機会の喪失リスク**は高い。
読者は「緊急度が高い = 確実に起きる」と誤読しないよう注意されたい。

### リスク相互作用マッピングについて

因果矢印は**定性的**であり、強度や確率は示されていない。
**「因果の存在 ≠ 必然的連鎖」** である点に注意。
すべてのリスクが連鎖して必然的に破滅に至るわけではない。

### 本フレームワークの役割
| ❌ しないこと | ✅ すること |
|---------------|-------------|
| 起きる/起きないの予言 | 相対的危機度の比較 |
| 正確な発生確率の提示 | 優先順位の判断支援 |
| 科学的証明としての使用 | 思考整理・議論の出発点 |

### 適切な用途と不適切な用途

| ✅ 適切な用途 | ❌ 不適切な用途 |
|---------------|----------------|
| 思考整理 | 発生確率の提示 |
| 議論の起点 | 年次予測の根拠 |
| 優先順位の仮置き | 政策決定の直接材料 |
| SF設定の現実アンカー | 科学的確定値としての引用 |

**本表は「未来を当てに行く道具」ではなく「当てに行きたくなる衝動を抑える道具」として設計されている。**
