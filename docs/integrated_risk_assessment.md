## 評価フレームワーク（固定ルール）

### 脅威度の計算式
```
脅威度 = 0.30×影響規模 + 0.30×不可逆性 + 0.25×致死性 + 0.15×近接性
```

### 緊急度の計算式
```
緊急度 = 脅威度 × (1 - 認知度/10) × (1 - 政策対応度/10)
```

## 腸内細菌操作リスクの位置づけ

### なぜ合成生物学バンドルの子リスクか

| 観点 | 内容 |
|------|------|
| 技術基盤 | CRISPR + AI統合（細菌編集）が共通 |
| 対象 | 「生態系」の一種（人体内生態系） |
| 構造 | 生態系ハイジャックと類似（多様性→モノカルチャー→脆弱性） |

### 独自性

| 観点 | 他の子リスクとの違い |
|------|---------------------|
| 対象 | 人体内（バイオテロ：人体外、生態系：環境） |
| 可視性 | **最も見えにくい**（外見変化なし、検出困難） |
| 規制状況 | **最も規制されていない**（サプリメント扱い） |
| 認知 | **リスクとして認知されていない**（「良いもの」フレーミング） |

### A/P紛争との接続

腸内細菌操作は「見えないオーグメンテーション」として、A/P紛争の新たな戦線を開く可能性：

- 遺伝子強化：検出可能、議論あり
- BCI強化：検出可能、議論あり
- **腸内細菌強化：検出困難、議論なし** ← 最も「見えない」格差

## 発見事項

### パターン1：合成生物学バンドルの強化

腸内細菌操作の追加により：

| 変化 | 内容 |
|------|------|
| バンドル内リスク数 | 3 → 4 |
| 緊急度平均 | 71 → 68（微減だが依然最高） |
| カバー範囲 | 人体外生態系 → 人体内生態系も含む |

### パターン2：「見えないリスク」の発見

| リスク | 「見えにくさ」の種類 |
|--------|---------------------|
| 複合毒性システム | 研究が存在しない |
| 腸内細菌操作 🆕 | 「良いもの」として認知されている |
| ミーム兵器 | 操作されていることに気づかない |

→ **「名前がない」だけでなく「見えない」ことが最大のリスク要因**

| 順位 | リスク | 緊急度 | 共通点 |

| 8 | 神経的自由意志喪失 | 60 | 認知度・政策対応度が極低 |

→ **「名前がついていない」リスクが集中**

## 信憑性に関する注記

### 新規リスク（腸内細菌操作）の確度

| 確度 | 内容 |
|------|------|
| A（高） | 腸脳相関の科学的事実 |
| B（中） | 意図的操作の技術的可能性 |
| C（低） | テロシナリオ、パンデミックシナリオ |

**本リスクは「research gap」が最も深刻な領域の一つ。リスクとしての研究が存在しない。**

### 本フレームワークの役割

| ❌ しないこと | ✅ すること |
|---------------|-------------|
| 起きる/起きないの予言 | 相対的危機度の比較 |
| 正確な発生確率の提示 | 優先順位の判断支援 |
| 科学的証明としての使用 | 思考整理・議論の出発点 |
