## 脅威度の4要素分解

### 影響規模：90

- パンデミック規模の病原体放出は全人類に波及
- COVID-19の経験から、初期封じ込め失敗時の指数関数的拡大は実証済み
- グローバル化された物流・人流により、局所発生→全球拡散の時間は短縮

### 不可逆性：85

- 病原体自体は時間経過で制御可能（ワクチン開発、集団免疫）
- ただし、人的被害・社会構造への影響は不可逆
- 「バイオテロは可能」という認知が広まること自体が不可逆

### 致死性：95

- COVID-19より高感染・高致死の病原体製造の技術障壁が急速に低下
- 数千万〜数億人規模の死傷潜在性
- 医療システムの崩壊による二次的死亡

### 近接性：70

- 臨界点：2032-2038年
- DIYバイオ施設は既に100カ国以上に存在
- DNA合成サービスのスクリーニング機能が形骸化進行中
- AI遺伝子設計ツールのクラウド汎用化が2027年頃に予測

## 技術発展ロードマップ

### Phase 1: 2025-2030（設計民主化期）

**技術進展**：
- AI遺伝子設計ツール（クラウドベース）が汎用化
- DNA合成サービスのスクリーニング機能が形骸化リスク増加
- CRISPR精度（オフターゲット率）が1/10000以下に低下

**社会状況**：
- 「デザイナーベビー」技術の非公式実装が始まる
- 規制の国別ギャップが拡大（中国・東南アジア vs 欧米）

### Phase 2: 2030-2040（統合・加速期）【臨界期】

**技術進展**：
- AI-CRISPR統合プラットフォームが実用化
- 「マルチエディット」（複数遺伝子同時改変）が自動化
- リアルタイム検証フィードバック（in silico → in vivo → in vitro ループ）

**リスク顕在化**：
- SARS-CoV-2超強化体の製造テンプレート化
- エアロゾル感染率の自動最適化が可能に
- **個人レベルでのパンデミック発生能力が成立**

### Phase 3: 2040-2050（分化・市場化期）

**技術進展**：
- オープンソース遺伝子デザインライブラリの標準化
- 個人向けCRISPRキット市場化（プリンタ並みの価格化）

**状況**：
- バイオテロの「民主化」が完了
- 防御技術との軍拡競争状態

## 認知度が極めて低い理由

### 認知度：0.8（14リスク中最低レベル）

| 層 | 状況 |
|-----|------|
| 学術界 | テロ研究はサイバー・核中心。バイオテロの「民主化」モデルは未体系化 |
| 政策層 | 生物兵器禁止条約（BWC）は国家対象。個人レベルの脅威は想定外 |
| メディア | パンデミック報道は「自然発生」前提。意図的放出シナリオは陰謀論扱い |
| 一般大衆 | SF的認知のみ。「自分にも作れる」という現実感なし |

### 認知されない構造的理由

1. **既存フレームワークとの不整合** - テロ対策は国家・組織対象。個人は想定外
2. **技術進歩の過小評価** - 「そんなに簡単ではない」という専門家バイアス
3. **心理的回避** - 考えたくない脅威は認知されない
4. **情報セキュリティ上の抑制** - 詳細な議論自体がリスクという判断

| 観点 | バイオテロ民主化 | 生態系ハイジャック |
|------|-----------------|-------------------|

| 意図 | 悪意あり（必須） | なくても発生 |

| 封鎖・隔離 | 可能（困難だが） | 不可能 |

### 共通点

- 根本技術が同一（CRISPR + AI統合）
- 技術障壁の低下が同時進行
- 規制の国別ギャップが脆弱性
- 認知度・政策対応度が極めて低い

## 信憑性に関する注記

### 本リスク評価の確度

| 確度 | 内容 |
|------|------|
| A（高） | 技術的実現可能性 - 既存技術の延長で可能 |
| B（中） | 臨界点の時期 - 2032-2038年は推定 |
| C（低） | 実際の発生シナリオ - 推測的 |

### 本評価の限界

- 「いつ起きるか」は予測不能
- 「誰が起こすか」も予測不能
- 「起きない可能性」も排除できない

**本評価は「起きる」と断定するものではなく、「準備すべきか」を判断するための思考ツールである。**
