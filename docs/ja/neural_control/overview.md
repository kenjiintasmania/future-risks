## 1. バンドル概要

### 定義

> **神経支配脅威バンドル**：BCI（Brain-Computer Interface）および関連神経技術の発展により、人間の神経系が外部からの意図的操作または非意図的被害の対象となるリスク群。

## 2. 親リスク：BCIハッキング

### 基本定義

| 項目 | 内容 |
|------|------|
| リスク名 | BCIハッキング（BCI Hacking / Brainjacking） |
| 定義 | 脳に埋め込まれた神経デバイス（BCI/DBS）への不正アクセス・操作 |
| 根本技術 | BCI + AI統合、深部脳刺激（DBS） |
| 臨界点 | 2030-2035年（商用BCI普及期） |

### 評価スコア

| 指標 | 値 | 根拠 |
|------|-----|------|
| 脅威度 | 73.50 | 影響75、不可逆85、致死60、近接70 |
| 認知度 | 3.0 | 学術論文あり、一般認知は低い |
| 政策対応度 | 1.5 | NeuroRights議論開始、実装なし |
| 緊急度 | **51**（11位） | 中脅威×低認知 |

### 現在の脆弱性（2025年時点）

#### Installed Base

| カテゴリ | 累計数（2025年） | 備考 |
|---------|-----------------|------|
| **DBS（深部脳刺激）** | **244,000人+** | パーキンソン病等、最大規模 |
| 人工内耳 | 1,000,000人+ | 聴覚補助 |
| 侵襲型BCI臨床試験 | 50-100人 | Neuralink 12人、他社含む |

#### 技術的脆弱性

| 脆弱性 | 詳細 |
|--------|------|
| **通信の暗号化欠如** | プログラマーのover-the-air送信が「暗号化なし・認証なし」 |
| **汎用タブレット使用** | iPad/iPhone経由の制御→ソフトウェアレベルの攻撃面 |
| **EMI脆弱性** | MAUDE database 446件の被害報告（2002-2019年） |

## 3. 子リスク構造

### 子リスク①：神経的自由意志喪失

| 項目 | 内容 |
|------|------|
| 定義 | BCI/DBS技術により個人のAgencyが外部から操作される状態 |
| 脅威度 | 75.50 |
| 緊急度 | 60（8位）🆘 |
| 脅威行為者 | Puppeteer（個人攻撃）、Sovereign（国家統制） |

#### 名称に関する注記

「自由意志（Free Will）」は哲学的に未解決の概念。本プロジェクトでは：
- **名称**：「神経的自由意志喪失」を維持（思考過程の記録）
- **分析枠組み**：「Agency（行為主体性）侵害」を使用
- **対策**：修正NeuroRights原則3「行為主体性の保護」に接続

→ 詳細は [神経的自由意志喪失：詳細](./neurological_free_will_loss.md)  
→ 哲学的分析は [自由意志・Agency・自己の分析](./free_will_agency_self_analysis.md)

### 子リスク②：複合毒性システム

| 項目 | 内容 |
|------|------|
| 定義 | マイクロプラスチック＋BCI/DBSの相互作用による予測不能な長期的神経毒性 |
| 脅威度 | 73.50 |
| 緊急度 | 66（4位）🆘 |
| 特徴 | **research gapが最も深刻**（研究が存在しない） |

#### なぜ緊急度4位か

```
緊急度 = 脅威度 × (1 - 認知度/10) × (1 - 政策対応度/10)
       = 73.50 × (1 - 0.05) × (1 - 0.05)
       = 73.50 × 0.95 × 0.95
       = 66
```

脅威度は中程度だが、**認知度0.5・政策対応度0.5という極低値**が緊急度を押し上げている。

→ 詳細は [複合毒性システム：詳細](./composite_toxicity_system.md)

### 子リスク比較

| 項目 | 神経的自由意志喪失 | 複合毒性システム |
|------|-------------------|-----------------|
| 脅威行為者 | Puppeteer / Sovereign | **なし**（構造的リスク） |
| 意図性 | **意図的攻撃** | **非意図的蓄積** |
| 時間軸 | 即時〜短期 | **長期（10-30年）** |
| 検出 | 困難 | **極めて困難** |
| 対策方向 | NeuroRights（規範） | **研究促進**（問いを立てる） |

## 4. 脅威行為者

### Puppeteer（操り師）：個人レベル

| 項目 | 内容 |
|------|------|
| 動機 | 個人支配、復讐、犯罪（恐喝、詐欺） |
| 手段 | 単独BCI侵害、Brainjacking、パラメータ改ざん |
| 対象 | 特定個人（ex-パートナー、ビジネス上の敵） |
| 対策 | デバイスセキュリティ、法的保護 |

### Sovereign（主権者）：国家レベル

| 項目 | 内容 |
|------|------|
| 動機 | 政治的統制、反体制派抑圧、国民管理 |
| 手段 | 国家レベルの神経監視・介入インフラ |
| 対象 | 特定集団または全国民 |
| 対策 | 国際的NeuroRights規範、透明性要件 |

### 構造的リスク（複合毒性）

| 項目 | 内容 |
|------|------|
| 動機 | **なし**（非意図的） |
| 手段 | 環境汚染＋医療技術の組み合わせ |
| 対象 | BCI装着者全員 |
| 対策 | 研究促進、長期モニタリング |

## 5. 他バンドルとの関係

### 基盤リスク層との接続

```
マイクロプラスチック（基盤リスク、77.50）
    │
    ├─→ 合成生物学B
    │      └→ 海洋・土壌生態系への影響
    │
    ├─→ 認識論崩壊B
    │      └→ 認知能力低下→情報処理能力低下
    │
    └─→ 神経支配B ★
           └→ **複合毒性システム（緊急度4位）**
```

### A/P紛争との接続

```
A/P紛争（ハブリスク、82.75）
    │
    ├─→ 合成生物学B経由
    │      └→ 遺伝子強化
    │
    └─→ 神経支配B経由 ★
           └→ BCI強化（認知能力、感覚拡張）
           └→ 強化者 vs 非強化者の分断
```

### 認識論崩壊Bとの接続

| 接続点 | 内容 |
|--------|------|
| 基盤0（委譲の自覚） | 両バンドル共通の「最後の砦」 |
| 情報処理能力 | マイクロプラ→認知低下→ミーム兵器への脆弱性増加 |
| 検出困難性 | 両バンドルとも「操作されていることに気づかない」問題 |
